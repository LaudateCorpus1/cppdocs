Function Documentation¶

std::shared_ptr<script::Module> torch::jit::load(const std::string &filename, c10::optional<c10::Device> device = c10::nullopt, script::ExtraFilesMap &extra_files = default_extra_files)

Loads a serialized script::Module from the given filename.

The file stored at the location given in filename must contain a serialized script::Module, exported either via ScriptModule.save() in Python or torch::jit::ExportModule in C++.
