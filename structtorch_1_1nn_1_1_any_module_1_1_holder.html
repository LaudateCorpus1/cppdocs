Detailed Description

template<typename ModuleType, typename... ArgumentTypes>
struct torch::nn::AnyModule::Holder< ModuleType, ArgumentTypes >

The dynamic type of the object stored in the AnyModule. It contains the concrete instance to which all calls are forwarded. It is parameterized over the concrete type of the module, and the types of the arguments the module takes in its forward() method.
