Detailed Description

A simplified implementation of std::any which stores a type erased object, whose concrete value can be retrieved at runtime by checking if the typeid() of a requested type matches the typeid() of the object stored. It is simplified in that it does not handle copying, as we do not require it for our use cases. Moves are sufficient.
