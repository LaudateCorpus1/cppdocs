Detailed Description

template<size_t D, typename T = int64_t>
class torch::ExpandingArray< D, T >

A utility class that accepts either a container of D-many values, or a single value, which is internally repeated D times. This is useful to represent parameters that are multidimensional, but often equally sized in all dimensions. For example, the kernel size of a 2D convolution has an x and y length, but x and y are often equal. In such a case you could just pass 3 to an ExpandingArray<2> and it would "expand" to {3, 3}.
